Añadidas esta dependencia al pom.xml de forma manual:

JSON Web Token Support For The JVM
<!-- https://mvnrepository.com/artifact/io.jsonwebtoken/jjwt -->
<dependency>
    <groupId>io.jsonwebtoken</groupId>
    <artifactId>jjwt</artifactId>
    <version>0.9.1</version>
</dependency>

JAXB API:
<!-- https://mvnrepository.com/artifact/javax.xml.bind/jaxb-api -->
<dependency>
    <groupId>javax.xml.bind</groupId>
    <artifactId>jaxb-api</artifactId>
    <version>2.3.1</version>
</dependency>


CSRF
Is an attack vector that tricks a web browser into executing an unwanted action in an application to which 
a user is logged in.
Es un vector de ataque que engaña a un navegador web para que ejecute una acción no deseada en una aplicación 
en la que un usuario ha iniciado sesión
